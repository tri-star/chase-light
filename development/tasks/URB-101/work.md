# フィード詳細画面追加 作業計画

## 作業概要
フィード一覧画面から詳細画面へ遷移する機能と、フィード詳細画面を実装します。

## 作業内容

### 1. 実装するファイル
- **ページコンポーネント**
  - `/packages/app/src/pages/feeds/[id].vue` - 詳細ページのエントリーポイント
  - `/packages/app/src/components/pages/feed-detail/FeedDetailPage.vue` - 詳細ページのメインコンポーネント
  
- **API実装**
  - `/packages/app/src/server/api/feeds/[id].get.ts` - フィード詳細取得API

### 2. 修正するファイル
- `/packages/app/src/components/pages/feed-list/parts/FeedListTable.vue` - フィード名をリンクに変更

### 3. 実装の流れ
1. フィード一覧画面でのリンク実装
2. 詳細ページのルーティング実装
3. 詳細ページのコンポーネント実装（仮データで動作確認）
4. フィード詳細取得APIの実装
5. 仮データをAPI呼び出しに置き換え

### 4. 詳細ページのUI構成
